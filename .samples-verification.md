## What was fixed (samples)
- **Middleware removed everywhere**: all flow files are now byte-identical to the canonical middleware-free,
  type-exporting snippet (app + backend); Go (chi/echo/gin) dropped `&middleware.Middleware{}` +
  `ai.WithUse(&Retry{})`; Dart dropped `RetryPlugin()` + `use:[retry()]`. Framework middleware preserved.
- **Full-stack type sharing**: UIs import `Recipe`/`PartialRecipe`/`BargainChefInput` and use typed `streamFlow`.
- **Build breakers fixed**: react-vite `vite-env.d.ts`; Go pointer pattern; django SSE `model_dump`;
  unpublished `@genkit-ai/fetch`/`@genkit-ai/middleware` pins.
- **Doc-driven fixes mirrored**: Next.js App Router generic, Fastify setup, Remix/TanStack scaffolders + APIs.
- **tanstack-start/api-route**: added the missing UI route.
- **Dart Shelf**: the repo-wide `bin/` ignore was hiding the sample source — un-ignored; it now ships its code.
- **READMEs**: stale `frontends/`/`backends/` paths, absolute `/Users/...` paths, and `genkit.dev/docs/frameworks/...`
  URLs corrected across all samples.
