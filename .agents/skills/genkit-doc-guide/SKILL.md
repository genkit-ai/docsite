# Genkit Documentation Guidance

This skill provides expert guidance for maintaining Genkit documentation.

## Usage

When authoring or reviewing documentation, consult the [Authoring Guidance](references/guidance.md).

## Workflow

1. **Verify** the file location is `src/content/docs/docs/`.
2. **Review** frontmatter for `supportedLanguages` and `isLanguageAgnostic` compliance.
3. **Check** content for appropriate `Lang` component usage.
4. **Validate** links against the "Internal Docs Links" section in the guidance.
5. **Ensure** code examples are consistent across languages where applicable.

If you encounter issues during your process, refer back to the [Authoring Guidance](references/guidance.md).
