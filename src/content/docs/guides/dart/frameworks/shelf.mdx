Use this guide when you're starting from a Dart server and want to expose a Genkit flow through Shelf.

## Before you begin

- Dart SDK 3.10.0 or later

This guide assumes you're already familiar with building Dart server applications.

## Create a project

Create a new Dart console app:

```bash
dart create -t console-simple my_genkit_shelf
cd my_genkit_shelf
```

## Project layout

<FileTree>

- pubspec.yaml
- bin
  - **my_genkit_shelf.dart**
  - my_genkit_shelf.g.dart generated schema code

</FileTree>

## Install Genkit packages

Install the Genkit CLI:

```bash
curl -sL cli.genkit.dev | bash
```

Then install the packages you need in your project:

```bash
dart pub add genkit genkit_google_genai genkit_shelf schemantic shelf shelf_router dev:build_runner
```

## Configure your model API key

This guide uses the Gemini API from Google AI Studio.

```bash
export GEMINI_API_KEY=<your API key>
```

## Create your application

Replace `bin/my_genkit_shelf.dart` with the same recipe flow used in the main quickstart, then expose it through Shelf:

```dart title="bin/my_genkit_shelf.dart"
import 'dart:io';

import 'package:genkit/genkit.dart';
import 'package:genkit_google_genai/genkit_google_genai.dart';
import 'package:genkit_shelf/genkit_shelf.dart';
import 'package:schemantic/schemantic.dart';
import 'package:shelf/shelf.dart';
import 'package:shelf/shelf_io.dart' as io;
import 'package:shelf_router/shelf_router.dart';

part 'my_genkit_shelf.g.dart';

@Schema()
abstract class $RecipeInput {
  @Field(description: 'Main ingredient or cuisine type')
  String get ingredient;

  @Field(description: 'Any dietary restrictions')
  String? get dietaryRestrictions;
}

@Schema()
abstract class $Recipe {
  String get title;
  String get description;
  String get prepTime;
  String get cookTime;
  int get servings;
  List<String> get ingredients;
  List<String> get instructions;
  List<String>? get tips;
}

void main() async {
  final ai = Genkit(plugins: [googleAI()]);

  final recipeGeneratorFlow = ai.defineFlow(
    name: 'recipeGeneratorFlow',
    inputSchema: RecipeInput.$schema,
    outputSchema: Recipe.$schema,
    fn: (input, _) async {
      final dietaryRestrictions = input.dietaryRestrictions ?? 'none';
      final prompt = 'Create a recipe with the following requirements:\n'
          'Main ingredient: ${input.ingredient}\n'
          'Dietary restrictions: $dietaryRestrictions';

      final response = await ai.generate(
        model: googleAI.gemini('gemini-2.5-flash'),
        config: GeminiOptions(temperature: 0.8),
        prompt: prompt,
        outputSchema: Recipe.$schema,
      );

      if (response.output == null) {
        throw Exception('Failed to generate recipe');
      }

      return response.output!;
    },
  );

  final router = Router();
  router.post('/recipeGeneratorFlow', shelfHandler(recipeGeneratorFlow));

  final handler = const Pipeline().addMiddleware(logRequests()).addHandler(router.call);
  final server = await io.serve(handler, InternetAddress.anyIPv4, 8080);
  print('Server running on http://localhost:${server.port}');
}
```

## Run your application

First, generate the schema code:

```bash
dart run build_runner build
```

Then run your app:

```bash
dart run
```

You can also launch it through the Developer UI:

```bash
genkit start -- dart run
```

## Test your flow

With the server running, call your flow over HTTP:

```bash
curl -X POST http://localhost:8080/recipeGeneratorFlow \
  -H "Content-Type: application/json" \
  -d '{"data":{"ingredient":"avocado","dietaryRestrictions":"vegetarian"}}'
```

You should receive a structured recipe in the response.

## Serving models

You can also expose a model directly with `shelfHandler` so other Genkit apps can consume it as a remote model. For example:

```dart title="bin/my_genkit_shelf.dart"
import 'dart:io';

import 'package:genkit_google_genai/genkit_google_genai.dart';
import 'package:genkit_shelf/genkit_shelf.dart';
import 'package:shelf/shelf_io.dart' as io;
import 'package:shelf_router/shelf_router.dart';

void main() async {
  final geminiApi = googleAI();
  final geminiFlash = geminiApi.model('gemini-2.5-flash');

  final router = Router();
  router.post('/googleai/gemini-2.5-flash', shelfHandler(geminiFlash));

  final server = await io.serve(router.call, InternetAddress.anyIPv4, 8080);
  print('Server running on http://localhost:${server.port}');
}
```
