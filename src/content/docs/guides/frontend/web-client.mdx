Use this guide when you have a Genkit backend exposing flows as HTTP endpoints and want to call them from a web frontend.

This pattern works with any JavaScript or TypeScript web app — React, Vue, Angular, plain JS, or anything else. Your backend can be written in any language — Node.js, Go, Python, or Dart.

## Before you begin

- A Genkit backend with at least one flow exposed as an HTTP endpoint. See [Choose your stack](/docs/choose-your-stack) to set one up.
- A web frontend project using JavaScript or TypeScript

## Install the Genkit client library

```bash
npm install genkit
```

## Call a flow

### Non-streaming

Use `runFlow` for flows that return a single, complete response:

```typescript
import { runFlow } from 'genkit/beta/client';

// Replace this URL with the address of your Genkit backend
const result = await runFlow({
  url: 'http://localhost:8080/recipeGeneratorFlow',
  input: {
    ingredient: 'avocado',
    dietaryRestrictions: 'vegetarian',
  },
});

console.log(result);
```

### Streaming

Use `streamFlow` for flows that stream responses incrementally:

```typescript
import { streamFlow } from 'genkit/beta/client';

// Replace this URL with the address of your Genkit backend
const result = streamFlow({
  url: 'http://localhost:8080/recipeGeneratorFlow',
  input: {
    ingredient: 'avocado',
    dietaryRestrictions: 'vegetarian',
  },
});

for await (const chunk of result.stream) {
  console.log('chunk:', chunk);
}

const output = await result.output;
console.log('final:', output);
```

## Authentication

If your deployed flow requires authentication, pass headers with your request:

```typescript
const result = await runFlow({
  url: 'https://your-backend.example.com/recipeGeneratorFlow',
  headers: {
    Authorization: 'Bearer your-token-here',
  },
  input: { ingredient: 'avocado' },
});
```

## When deploying with Firebase

When your backend is deployed to [Cloud Functions for Firebase](/guides/deployment/firebase), use the Firebase callable functions client library instead:

```typescript
import { getFunctions, httpsCallable } from 'firebase/functions';

const functions = getFunctions();
const recipeGeneratorFlow = httpsCallable(functions, 'recipeGeneratorFlow');

// For streaming responses
const { stream, data } = await recipeGeneratorFlow.stream({
  ingredient: 'avocado',
  dietaryRestrictions: 'vegetarian',
});

for await (const chunk of stream) {
  console.log('chunk:', chunk);
}

const result = await data;
console.log('final:', result);
```

See the [Firebase callable functions documentation](https://firebase.google.com/docs/functions/callable) for full details.
