## After your server is running

Once your backend exposes flows as HTTP endpoints, connect a frontend:

- [Web client](/docs/web-client) — call flows from any JavaScript/TypeScript web app
- [Flutter](/docs/app-frameworks/flutter) — call flows from a Flutter mobile, desktop, or web app
- Or use any of the [app integration guides](/docs/app-frameworks/overview), full-stack frameworks like Next.js, SvelteKit, Nuxt, and others can also consume a standalone Genkit backend
