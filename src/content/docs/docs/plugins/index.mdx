This page provides an overview of all available plugins for Genkit JS. Plugins extend Genkit's functionality by providing integrations with various services, models, and databases.

<CardGrid>
  <Card title="Google AI">
    The Google Generative AI plugin provides interfaces to Google's Gemini models through the Gemini API. It allows you to use Gemini models for text generation and embeddings.
    <br />
    <a href="./plugins/google-genai">View plugin info</a>
  </Card>
  <Card title="Vertex AI">
    The Vertex AI plugin provides interfaces to several AI services from Google Cloud, including Gemini text generation, Imagen image generation, text and multimodal embedding generation, evaluation metrics, and Vector Search.
    <br />
    <a href="./plugins/vertex-ai">View plugin info</a>
  </Card>
  <Card title="Ollama">
    The Ollama plugin provides interfaces to any of the local LLMs supported by Ollama.
    <br />
    <a href="./plugins/ollama">View plugin info</a>
  </Card>
  <Card title="Firebase">
    The Firebase plugin provides integrations with Firebase services, so you can build intelligent and scalable AI applications. Key features include Firestore Vector Store for indexing and retrieval with vector embeddings, and telemetry export to Google's Cloud operations suite.
    <br />
    <a href="./plugins/firebase">View plugin info</a>
  </Card>
  <Card title="LanceDB">
    The LanceDB plugin provides indexer and retriever implementations that use LanceDB, an open-source vector database for AI applications.
    <br />
    <a href="./plugins/lancedb">View plugin info</a>
  </Card>
  <Card title="Astra DB">
    The Astra DB plugin integrates Astra DB into the Genkit framework as an indexer and retriever. You can efficiently embed, index, and retrieve data within your Genkit applications.
    <br />
    <a href="./plugins/astra-db">View plugin info</a>
  </Card>
  <Card title="Auth0 AI">
    The Auth0 plugin enables you to build secure AI-powered applications using Auth0 and Okta FGA.
    <br />
    <a href="./plugins/auth0">View plugin info</a>
  </Card>
  <Card title="ChromaDB">
    The Chroma plugin provides indexer and retriever implementations that use the Chroma vector database in client/server mode.
    <br />
    <a href="./plugins/chroma">View plugin info</a>
  </Card>
  <Card title="Neo4j">
    The Neo4j plugin integrates Neo4j graph databases into the Genkit framework. You can leverage knowledge graphs and semantic search for building advanced AI applications, particularly for RAG scenarios where contextual information from a knowledge graph is crucial.
    <br />
    <a href="./plugins/neo4j">View plugin info</a>
  </Card>
  <Card title="pgvector" icon="document">
    The pgvector template is an example PostgreSQL and pgvector retriever implementation. You can use the provided
    examples as a starting point and modify them to work with your database schema.
    <br />
    <a href="./plugins/pgvector">View template</a>
  </Card>
  <Card title="Pinecone">
    The Pinecone plugin provides indexer and retriever implementations that use the Pinecone cloud vector database.
    <br />
    <a href="./plugins/pinecone">View plugin info</a>
  </Card>
</CardGrid>
