This page lists third-party plugins for Genkit that are built and maintained by Firebase or
our partners.

<CardGrid>
  <Card title="Neo4j">
    The Neo4j plugin integrates the{' '}
    <a class="inline-flex no-underline" href="https://neo4j.com/product/neo4j-graph-database/" target="_blank">
      Neo4j graph databases
      <Icon name="external" />
    </a>{' '}
    into the Genkit framework. You can leverage knowledge graphs and semantic search for building advanced AI
    applications, particularly for RAG scenarios where contextual information from a knowledge graph is crucial.
    <br />
    <a class="inline-flex no-underline" href="https://github.com/neo4j-partners/genkitx-neo4j" target="_blank">
      View plugin info
      <Icon name="external" />
    </a>
  </Card>
  <Card title="DataStax Astra DB">
    The Astra DB plugin integrates{' '}
    <a class="inline-flex no-underline" href="https://www.datastax.com/products/datastax-astra" target="_blank">
      Astra DB
      <Icon name="external" />
    </a>{' '}
    into the Genkit framework as an indexer and retriever. You can efficiently embed, index, and retrieve data within
    your Genkit applications.
    <br />
    <a class="inline-flex no-underline" href="https://github.com/datastax/genkitx-astra-db" target="_blank">
      View plugin info
      <Icon name="external" />
    </a>
  </Card>

      <Icon name="external" />
    </a>{' '}

    <a href="../chroma">View plugin info</a>
