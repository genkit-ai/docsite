This page lists third-party plugins for Genkit that are built and maintained by Firebase or
our partners.

<CardGrid stagger>
	<Card title="Pinecone">
		The Pinecone plugin provides indexer and retriever implementations that use the <a class="inline-flex no-underline"href="https://www.pinecone.io/product/" target="_blank">Pinecone<Icon name="external" /></a> cloud vector database.
        <br/><a href="../pinecone">View plugin info</a>
	</Card>
	<Card title="Ollama">
		The Ollama plugin provides interfaces to any of the local LLMs supported by
        <a class="inline-flex no-underline"href="https://ollama.com" target="_blank">Ollama<Icon name="external" /></a>.
        <br/><a href="../ollama">View plugin info</a>
	</Card>
  <Card title="pgvector" icon="document">
		The pgvector template is an example PostgreSQL and pgvector retriever implementation. You
        can use the provided examples as a starting point and modify them to work with your database
        schema.
        <br/><a href="../pgvector">View template</a>
	</Card>
</CardGrid>
